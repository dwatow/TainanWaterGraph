# 台南年度用戶端水質檢測資料查詢

資料來源: [用戶端水質數據](https://www.water.gov.tw/dist6/WaterQualityClient?nodeId=6701)

## 說明

在此程式只是將這些數據全部圖表化，並且可以用地區關鍵字，查找全台南的用戶端水質資料
